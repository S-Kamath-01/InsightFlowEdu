# Security Policy

## Supported Versions
This repository is maintained as a portfolio= project. Security fixes will be applied to the latest published version of the repository.

## Reporting a Vulnerability
If you find a security issue, do not post exploit details publicly.

Please report:
- the affected file or endpoint
- a short description of the issue
- the impact you observed
- any safe reproduction steps

## Current Security Notes
- The backend includes demo-friendly authentication and permissive local development settings.
- Do not reuse the sample credentials or sample database passwords in production.
- Treat all credentials shown in setup docs as placeholders unless you are running the local demo environment.
